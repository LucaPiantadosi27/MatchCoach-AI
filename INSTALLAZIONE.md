# 📱 Guida all'Installazione di MatchCoach

MatchCoach è una Progressive Web App (PWA) che può essere installata su qualsiasi dispositivo mobile o desktop come un'app nativa.

## 📱 Android (Chrome)

### Requisiti
- Dispositivo Android
- Browser Google Chrome (versione recente)
- Connessione internet attiva

### Procedura di Installazione

1. **Apri MatchCoach con Chrome**
   - Avvia Google Chrome sul tuo telefono Android
   - Vai all'indirizzo web di MatchCoach

2. **Accedi al menu**
   - Tocca l'icona del menu (⋮) in alto a destra dello schermo

3. **Seleziona "Installa app"**
   - Nel menu a tendina, cerca e tocca l'opzione **"Installa app"** o **"Aggiungi a schermata Home"**
   - Potrebbe apparire anche un banner in basso con il pulsante "Installa"

4. **Conferma l'installazione**
   - Nella finestra di dialogo che appare, tocca **"Installa"**
   - Attendi qualche secondo per il completamento

5. **Trova l'app**
   - L'icona di MatchCoach apparirà nella schermata Home del tuo telefono
   - Puoi anche trovarla nel drawer delle app

### 🎯 Risultato
L'app si aprirà in modalità standalone, senza la barra del browser, proprio come un'app nativa!

## 🍎 iOS / iPhone (Safari)

### Requisiti
- iPhone o iPad con iOS 11.3 o superiore
- Browser Safari (predefinito)
- Connessione internet attiva

### Procedura di Installazione

1. **Apri MatchCoach con Safari**
   - Avvia Safari sul tuo iPhone/iPad
   - Vai all'indirizzo web di MatchCoach
   - **Importante**: Devi usare Safari, non Chrome o altri browser

2. **Apri il menu Condividi**
   - Tocca il pulsante **Condividi** (icona □↑) nella barra in basso
   - Se non vedi la barra, scorri leggermente verso il basso sulla pagina

3. **Trova "Aggiungi a Home"**
   - Nel menu che appare, scorri verso il basso
   - Cerca e tocca l'opzione **"Aggiungi a Home"** (icona con un +)
   - Se non la vedi subito, scorri le opzioni disponibili

4. **Personalizza il nome (opzionale)**
   - Apparirà una schermata di conferma
   - Puoi modificare il nome dell'app se vuoi
   - Il nome predefinito è "MatchCoach"

5. **Conferma l'aggiunta**
   - Tocca **"Aggiungi"** in alto a destra
   - L'icona verrà aggiunta alla schermata Home

### 🎯 Risultato
L'app apparirà sulla schermata Home come un'applicazione nativa iOS, con la sua icona e senza la barra di Safari!

## 💻 Desktop (Chrome/Edge)

### Requisiti
- Computer Windows, Mac o Linux
- Browser Chrome, Edge, o Brave
- Connessione internet attiva

### Procedura di Installazione

1. **Apri MatchCoach nel browser**
   - Avvia Chrome, Edge o Brave
   - Vai all'indirizzo web di MatchCoach

2. **Cerca l'icona di installazione**
   - Nella barra degli indirizzi, cerca l'icona di installazione (⊕ o 💻)
   - Di solito appare sul lato destro della barra

3. **Clicca su "Installa"**
   - Clicca sull'icona di installazione
   - Oppure vai al menu (⋮) → **"Installa MatchCoach..."**

4. **Conferma l'installazione**
   - Nella finestra di dialogo, clicca **"Installa"**
   - L'app verrà scaricata e installata

5. **Avvia l'app**
   - MatchCoach si aprirà in una finestra dedicata
   - Troverai l'icona nel menu Start (Windows) o nelle Applicazioni (Mac)

### 🎯 Risultato
L'app funzionerà come un programma desktop indipendente, senza le barre del browser!

## 🔧 Risoluzione Problemi

### ❓ Non vedo l'opzione "Installa app"

**Possibili cause:**
- Stai usando un browser non supportato
- L'app è già installata
- Il sito non è servito tramite HTTPS

**Soluzioni:**
- **Android**: Usa Google Chrome (non Samsung Internet o altri)
- **iOS**: Usa Safari (non Chrome o Firefox)
- **Desktop**: Usa Chrome, Edge o Brave
- Verifica di essere connesso a internet
- Prova a ricaricare la pagina (F5 o swipe down)

### ❓ L'app non funziona offline

**Soluzioni:**
- Assicurati di aver aperto l'app almeno una volta con connessione attiva
- Chiudi e riapri l'app
- Verifica che il service worker sia registrato (vedi console del browser)
- Cancella la cache e reinstalla l'app

### ❓ L'icona non appare sulla Home

**Soluzioni:**
- **Android**: Controlla nel drawer delle app
- **iOS**: Scorri le schermate Home, potrebbe essere nell'ultima pagina
- **Desktop**: Cerca nel menu Start o nella cartella Applicazioni
- Prova a reinstallare seguendo nuovamente i passaggi

### ❓ L'app si apre nel browser invece che standalone

**Soluzioni:**
- Disinstalla l'app e reinstallala
- Assicurati di aver confermato l'installazione correttamente
- Su iOS, verifica di aver usato "Aggiungi a Home" e non "Aggiungi ai Preferiti"

### ❓ Problemi di sincronizzazione dati

**Soluzioni:**
- Verifica di essere loggato con lo stesso account
- Controlla la connessione internet
- Forza la sincronizzazione ricaricando la pagina
- Se il problema persiste, contatta il supporto

## 📞 Supporto

Se hai ancora problemi con l'installazione:

1. Verifica di avere l'ultima versione del browser
2. Prova a disinstallare e reinstallare l'app
3. Controlla che il tuo dispositivo supporti le PWA
4. Consulta la sezione FAQ nell'app

## 🔄 Aggiornamenti

L'app si aggiorna automaticamente quando:
- Viene rilasciata una nuova versione
- Riapri l'app con connessione internet attiva
- Il service worker rileva modifiche

Non è necessario reinstallare l'app per ricevere gli aggiornamenti!
