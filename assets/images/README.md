# Immagini Background Campo

Questa cartella contiene le texture per gli sfondi del campo da gioco.

## File Richiesti

Carica i seguenti file in questa cartella:

### 1. parquet_chiaro.jpg
- **Descrizione:** Texture parquet chiaro per pavimento palazzetto
- **Formato:** JPG o PNG
- **Dimensioni consigliate:** 512x512px o superiore
- **Caratteristiche:** Tonalità legno chiaro, texture ripetibile

### 2. parquet_scuro.jpg
- **Descrizione:** Texture parquet scuro per pavimento palazzetto
- **Formato:** JPG o PNG
- **Dimensioni consigliate:** 512x512px o superiore
- **Caratteristiche:** Tonalità legno scuro, texture ripetibile

## Note Importanti

- Le immagini devono essere **seamless** (ripetibili senza giunture visibili)
- Formato consigliato: JPG per dimensioni file ridotte
- Le texture verranno ripetute automaticamente per coprire tutto il campo
- Se le immagini non sono presenti, verrà usato un parquet generato automaticamente

## Fallback

Se i file non sono presenti, l'app utilizzerà automaticamente una texture parquet generata via codice con:
- **Parquet Chiaro:** Colore base #D4A574
- **Parquet Scuro:** Colore base #8B6F47

## Dove Trovare Texture

Puoi trovare texture parquet gratuite su:
- [Textures.com](https://www.textures.com)
- [Freepik](https://www.freepik.com)
- [Unsplash](https://unsplash.com)

Cerca termini come: "wood floor texture", "parquet seamless", "basketball court floor"
